# Ad Injection Shortcodes

## `[before_actual_content]`

Manually places the "Before Content" ad block.

## `[after_actual_content]`

Manually places the "After Content" ad block.

### Usage
These shortcodes are typically used within templates or content where manual placement of the standard ad blocks is required.

```markdown
[before_actual_content]
... content ...
[after_actual_content]
```

### Attributes
These shortcodes do not accept any parameters.
