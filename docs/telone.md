# TelOne ADSL/Fibre/LTE

## `[telone]`

Displays TelOne internet packages.

### Usage
```markdown
[telone type="adsl"]
```

### Attributes

*   **type** (optional): The category of packages to display. Default is `All`.
    *   `adsl`: ADSL (ZiG)
    *   `adsl_usd`: ADSL (USD)
    *   `fibre`: Fibre
    *   `lte`: Blaze LTE (ZiG)
    *   `lte_usd`: Blaze LTE (USD)
    *   `wifi`: WIFI
    *   `vsat`: VSAT/LEO
    *   `usd`: USD Bonus Bundle
