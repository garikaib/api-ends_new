# Exchange Rates Shortcode

## `[zpc-latest-rates]`

**Aliases:** `[show-latest-rates]`

Displays the latest official and parallel market exchange rates (USD to ZiG/ZWL).

### Usage

```markdown
[zpc-latest-rates]
```

### Attributes
This shortcode does not accept any parameters.

### Output
Renders a table showing:
*   Official Bank Rate
*   Parallel Market Rate (if available)
*   OMIR (if available)
