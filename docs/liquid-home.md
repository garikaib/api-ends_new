# Liquid Home Packages

## `[liquid-home]`

Displays Liquid Home (formerly ZOL) internet packages.

### Usage
```markdown
[liquid-home type="fibre"]
```

### Attributes

*   **type** (optional): The category of packages to display. Default is `All`.
    *   `fibre`: FibroniX packages
    *   `vsat`: VSAT Packages
    *   `lte`: WibroniX
    *   `lte_usd`: WibroniX SpeeD USD
    *   `fibre_usd`: FibroniX SpeeD USD
