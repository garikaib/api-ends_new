# ZESA Tariffs

## `[zesa-tariffs]`

Displays ZESA electricity tariffs or the explanation of the stepped tariff system.

### Usage
```markdown
[zesa-tariffs type="tariffs"]
```

### Attributes

*   **type** (optional):
    *   `tariffs` (default): Displays the tariff table.
    *   `exp`: Displays the explanation of how tariffs work.
