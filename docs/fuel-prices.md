# Fuel Prices Shortcode

## `[zpc-latest-fuel]`

**Aliases:** `[show-latest-fuel-prices]`

Displays the latest regulated fuel prices (Petrol and Diesel) in USD and ZiG.

### Usage
```markdown
[zpc-latest-fuel]
```

### Attributes
This shortcode does not accept any parameters.

### Output
Renders a table with:
*   Commodity (Diesel 50 / Blend - E20)
*   Price in USD
*   Price in ZiG
