# NetOne Data Bundles

## `[netone-bundles]`

Displays NetOne mobile data and social bundles.

### Usage
```markdown
[netone-bundles type="All" filter="none"]
```

### Attributes

*   **type** (optional): Filter by bundle category. Default: `All`.
    *   Examples: `Data`, `Social`, `Voice` (depending on available data).
*   **filter** (optional): Additional filtering. Default: `none`.
