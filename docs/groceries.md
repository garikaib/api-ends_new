# Grocery Prices

## `[groceries]`

**Aliases:** `[groceries_new]`

Displays a table of basic grocery prices in Tuckshops (USD) and Supermarkets (ZiG estimate).

### Usage
```markdown
[groceries type="basic"]
```

### Attributes

*   **type** (optional): The category of groceries. Default is `basic`.
    *   `basic`: Basic commodities.
